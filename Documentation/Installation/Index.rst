============
Installation
============

This extension can be installed using Composer or from the TYPO3 Extension
Repository from within :guilabel:`Admin > Extensions`.

Composer
========

1. Run `composer require pixelant/demander` in the command line to install the
   extension.
2. Activate the extension in the :guilabel:`Admin > Extensions` or by running
   `typo3 extension:activate demander`.

Admin > Extensions
==================

Follow the instructions in the official documentation:
:ref:`t3install:install-an-extension-without-composer`
